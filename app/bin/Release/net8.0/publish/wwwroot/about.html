PTC License Tracker v1.0



About


PTC License Tracker v1.0 is an ASP.NET web application built in C# and JavaScript for OPCTurkey's internal use.
It helps engineers keep track of customer licenses and modify them, including when they expire and which product they're for.
It also provides fundemental data analysis and visualization abilities within.
